# Lodging System is built using Laravel 6.x

## Features:
* Branch/Store Management
* Room Management
* Booking and Reservation

### Add-on Features:
* Employee Management
* Accounting and Payroll
   * Attendance Management
   * Payroll and Reports
* Inventory and Warehousing
